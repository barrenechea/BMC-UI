USB route:

USB Modes: - Host - turns the USB_OTG port power on - use when you want to connect USB devices (like keyboard, mouse, USB drive, etc) through the USB_OTG port to supported modules - Device - the default mode, turns the USB_OTG power off - use in any other case - Flash - turns the module into flashing mode and sets the USB_OTG into device mode - use to flash the module using USB_OTG port
